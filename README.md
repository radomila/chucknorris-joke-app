# Chuck Norris Jokes web app

The main purpose of this web application is to display Chuck Norris jokes and allow users to add them to their favorites list. The web app contains several buttons, each representing a different action: 

- Receive one joke at a time
- Receive a new joke every 3 seconds
- Stop receiving new jokes
- Add to favourites 
- Clear the favourites list

## Tech Stack 
- ReactJS + Vite 
- TypeScript 
- Tailwind CSS
- Axios 
- Lucide React
